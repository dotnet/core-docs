# '&lt;name&gt;' cannot expose the underlying delegate type '&lt;delegatetype&gt;' of the event it is implementing outside the project through &lt;specifier&gt; '&lt;type&gt;'
The code exposes a private type outside the public class.  
  
 **Error ID:** BC30915  
  
## To correct this error  
  
-   Declare the type `Public`.  
  
     -or-  
  
-   Use another type.
