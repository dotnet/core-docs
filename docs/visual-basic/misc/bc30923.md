# '&lt;name&gt;' cannot expose type '&lt;type&gt;' used in the underlying delegate type '&lt;delegatetype&gt;' of the event it is implementing to &lt;specifier1&gt; '&lt;type1&gt;' through &lt;specifier2&gt; '&lt;type2&gt;'
The code exposes a private type outside the public class.  
  
 **Error ID:** BC30923  
  
## To correct this error  
  
-   Declare the type `Public` or use another type.
