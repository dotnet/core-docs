# Late-bound assignment to a field of value type '&lt;typename&gt;' is not valid when '&lt;name&gt;' is the result of a late-bound expression
You attempted a late-bound assignment that is not valid.  
  
## To correct this error  
  
-   Make the assignment early bound.
