# Constructors (C# Programming Guide)
Whenever a [class](../keywords/class--csharp-reference-.md) or [struct](../keywords/struct--csharp-reference-.md) is created, its constructor is called. A class or struct may have multiple constructors that take different arguments. Constructors enable the programmer to set default values, limit instantiation, and write code that is flexible and easy to read. For more information and examples, see [Using Constructors](../classes-and-structs/using-constructors--csharp-programming-guide-.md) and [Instance Constructors](../classes-and-structs/instance-constructors--csharp-programming-guide-.md).  
  
 If you do not provide a constructor for your object, C# will create one by default that instantiates the object and sets member variables to the default values as listed in [Default Values Table](../keywords/default-values-table--csharp-reference-.md). For more information and examples, see [Instance Constructors](../classes-and-structs/instance-constructors--csharp-programming-guide-.md).  
  
 Static classes and structs can also have constructors. For more information and examples, see [Static Constructors](../classes-and-structs/static-constructors--csharp-programming-guide-.md).
