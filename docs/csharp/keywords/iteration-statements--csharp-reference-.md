# Iteration Statements (C# Reference)
You can create loops by using the iteration statements. Iteration statements cause embedded statements to be executed a number of times, subject to the loop-termination criteria. These statements are executed in order, except when a [jump statement](../keywords/jump-statements--csharp-reference-.md) is encountered.  
  
 The following keywords are used in iteration statements:  
  
-   [do](../keywords/do--csharp-reference-.md)  
  
-   [for](../keywords/for--csharp-reference-.md)  
  
-   [foreach](../keywords/foreach--in--csharp-reference-.md)  
  
-   [in](../keywords/foreach--in--csharp-reference-.md)  
  
-   [while](../keywords/while--csharp-reference-.md)
