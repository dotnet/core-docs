# Types (C# Reference)
The C# typing system contains the following categories:  
  
-   [Value types](../keywords/value-types--csharp-reference-.md)  
  
-   [Reference types](../keywords/reference-types--csharp-reference-.md)  
  
-   [Pointer types](../unsafe-code-pointers/pointer-types--csharp-programming-guide-.md)  
  
 Variables that are value types store data, and those that are reference types store references to the actual data. Reference types are also referred to as objects. Pointer types can be used only in [unsafe](../keywords/unsafe--csharp-reference-.md) mode.  
  
 It is possible to convert a value type to a reference type, and back again to a value type, by using [boxing and unboxing](../types/boxing-and-unboxing--csharp-programming-guide-.md). With the exception of a boxed value type, you cannot convert a reference type to a value type.  
  
 This section also introduces [void](../keywords/void--csharp-reference-.md).  
  
 Value types are also nullable, which means they can store an additional non-value state. For more information, see [Nullable Types](../nullable-types/nullable-types--csharp-programming-guide-.md).
