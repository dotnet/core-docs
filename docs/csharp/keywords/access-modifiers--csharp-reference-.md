# Access Modifiers (C# Reference)
Access modifiers are keywords used to specify the declared accessibility of a member or a type. This section introduces the four access modifiers:  
  
-   [public](../keywords/public--csharp-reference-.md)  
  
-   [protected](../keywords/protected--csharp-reference-.md)  
  
-   [internal](../keywords/internal--csharp-reference-.md)  
  
-   [private](../keywords/private--csharp-reference-.md)  
  
 The following five accessibility levels can be specified using the access modifiers:  
  
 `public`: Access is not restricted.  
  
 `protected`: Access is limited to the containing class or types derived from the containing class.  
  
 `Internal`: Access is limited to the current assembly.  
  
 [protected internal](../classes-and-structs/access-modifiers--csharp-programming-guide-.md): Access is limited to the current assembly or types derived from the containing class.  
  
 `private`: Access is limited to the containing type.  
  
 This section also introduces the following:  
  
-   [Accessibility Levels](../keywords/accessibility-levels--csharp-reference-.md): Using the four access modifiers to declare five levels of accessibility.  
  
-   [Accessibility Domain](../keywords/accessibility-domain--csharp-reference-.md): Specifies where, in the program sections, a member can be referenced.  
  
-   [Restrictions on Using Accessibility Levels](../keywords/restrictions-on-using-accessibility-levels--csharp-reference-.md): A summary of the restrictions on using declared accessibility levels.
