# Query Keywords (C# Reference)
This section contains the contextual keywords used in query expressions.  
  
## In This Section  
  
|Clause|Description|  
|------------|-----------------|  
|[from](../keywords/from-clause--csharp-reference-.md)|Specifies a data source and a range variable (similar to an iteration variable).|  
|[where](../keywords/where-clause--csharp-reference-.md)|Filters source elements based on one or more Boolean expressions separated by logical AND and OR operators ( `&&` or `&#124;&#124;` ).|  
|[select](../keywords/select-clause--csharp-reference-.md)|Specifies the type and shape that the elements in the returned sequence will have when the query is executed.|  
|[group](../keywords/group-clause--csharp-reference-.md)|Groups query results according to a specified key value.|  
|[into](../keywords/into--csharp-reference-.md)|Provides an identifier that can serve as a reference to the results of a join, group or select clause.|  
|[orderby](../keywords/orderby-clause--csharp-reference-.md)|Sorts query results in ascending or descending order based on the default comparer for the element type.|  
|[join](../keywords/join-clause--csharp-reference-.md)|Joins two data sources based on an equality comparison between two specified matching criteria.|  
|[let](../keywords/let-clause--csharp-reference-.md)|Introduces a range variable to store sub-expression results in a query expression.|  
|[in](../keywords/in--csharp-reference-.md)|Contextual keyword in a [join](../keywords/join-clause--csharp-reference-.md) clause.|  
|[on](../keywords/on--csharp-reference-.md)|Contextual keyword in a [join](../keywords/join-clause--csharp-reference-.md) clause.|  
|[equals](../keywords/equals--csharp-reference-.md)|Contextual keyword in a [join](../keywords/join-clause--csharp-reference-.md) clause.|  
|[by](../keywords/by--csharp-reference-.md)|Contextual keyword in a [group](../keywords/group-clause--csharp-reference-.md) clause.|  
|[ascending](../keywords/ascending--csharp-reference-.md)|Contextual keyword in an [orderby](../keywords/orderby-clause--csharp-reference-.md) clause.|  
|[descending](../keywords/descending--csharp-reference-.md)|Contextual keyword in an [orderby](../keywords/orderby-clause--csharp-reference-.md) clause.|
