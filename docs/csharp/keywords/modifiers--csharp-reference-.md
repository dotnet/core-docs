# Modifiers (C# Reference)
Modifiers are used to modify declarations of types and type members. This section introduces the C# modifiers.  
  
|Modifier|Purpose|  
|--------------|-------------|  
|[Access Modifiers](../keywords/access-modifiers--csharp-reference-.md)<br /><br /> -   [public](../keywords/public--csharp-reference-.md)<br />-   [private](../keywords/private--csharp-reference-.md)<br />-   [internal](../keywords/internal--csharp-reference-.md)<br />-   [protected](../keywords/protected--csharp-reference-.md)|Specifies the declared accessibility of types and type members.|  
|[abstract](../keywords/abstract--csharp-reference-.md)|Indicates that a class is intended only to be a base class of other classes.|  
|[async](../keywords/async--csharp-reference-.md)|Indicates that the modified method, lambda expression, or anonymous method is asynchronous.|  
|[const](../keywords/const--csharp-reference-.md)|Specifies that the value of the field or the local variable cannot be modified.|  
|[event](../keywords/event--csharp-reference-.md)|Declares an event.|  
|[extern](../keywords/extern--csharp-reference-.md)|Indicates that the method is implemented externally.|  
|[new](../keywords/new--csharp-reference-.md)|Explicitly hides a member inherited from a base class.|  
|[override](../keywords/override--csharp-reference-.md)|Provides a new implementation of a virtual member inherited from a base class.|  
|[partial](../keywords/partial--type---csharp-reference-.md)|Defines partial classes, structs and methods throughout the same assembly.|  
|[readonly](../keywords/readonly--csharp-reference-.md)|Declares a field that can only be assigned values as part of the declaration or in a constructor in the same class.|  
|[sealed](../keywords/sealed--csharp-reference-.md)|Specifies that a class cannot be inherited.|  
|[static](../keywords/static--csharp-reference-.md)|Declares a member that belongs to the type itself instead of to a specific object.|  
|[unsafe](../keywords/unsafe--csharp-reference-.md)|Declares an unsafe context.|  
|[virtual](../keywords/virtual--csharp-reference-.md)|Declares a method or an accessor whose implementation can be changed by an overriding member in a derived class.|  
|[volatile](../keywords/volatile--csharp-reference-.md)|Indicates that a field can be modified in the program by something such as the operating system, the hardware, or a concurrently executing thread.|
