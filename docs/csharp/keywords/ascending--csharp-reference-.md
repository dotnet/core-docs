# ascending (C# Reference)
The `ascending` contextual keyword is used in the [orderby clause](../keywords/orderby-clause--csharp-reference-.md) in query expressions to specify that the sort order is from smallest to largest. Because `ascending` is the default sort order, you do not have to specify it.
