# new (C# Reference)
In C#, the `new` keyword can be used as an operator, a modifier, or a constraint.  
  
 [new Operator](../keywords/new-operator--csharp-reference-.md)  
 Used to create objects and invoke constructors.  
  
 [new Modifier](../keywords/new-modifier--csharp-reference-.md)  
 Used to hide an inherited member from a base class member.  
  
 [new Constraint](../keywords/new-constraint--csharp-reference-.md)  
 Used to restrict types that might be used as arguments for a type parameter in a generic declaration.
