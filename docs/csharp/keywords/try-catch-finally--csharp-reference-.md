# try-catch-finally (C# Reference)
A common usage of `catch` and `finally` together is to obtain and use resources in a `try` block, deal with exceptional circumstances in a `catch` block, and release the resources in the `finally` block.  
  
 For more information and examples on re-throwing exceptions, see [try-catch](../keywords/try-catch--csharp-reference-.md) and [Throwing Exceptions](../Topic/How%20to:%20Explicitly%20Throw%20Exceptions.md). For more information about the `finally` block, see [try-finally](../keywords/try-finally--csharp-reference-.md).
