# in (C# Reference)
The `in` contextual keyword is used in three contexts:  
  
-   [foreach](../keywords/foreach--in--csharp-reference-.md) statements  
  
-   [join clauses](../keywords/join-clause--csharp-reference-.md) in query expressions  
  
-   [generic type parameters](../keywords/in--generic-modifier---csharp-reference-.md) in generic interfaces and delegates.
