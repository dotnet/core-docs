# Operator Keywords (C# Reference)
Used to perform miscellaneous actions such as creating objects, checking the run-time type of an object, obtaining the size of a type, and other actions. This section introduces the following keywords:  
  
-   [as](../keywords/as--csharp-reference-.md) Converts an object to a compatible type.  
  
-   [await](../keywords/await--csharp-reference-.md) Suspends an async method until an awaited task is completed.  
  
-   [is](../keywords/is--csharp-reference-.md) Checks the run-time type of an object.  
  
-   [new](../keywords/new--csharp-reference-.md)  
  
    -   [new Operator](../keywords/new-operator--csharp-reference-.md) Creates objects.  
  
    -   [new Modifier](../keywords/new-modifier--csharp-reference-.md) Hides an inherited member.  
  
    -   [new Constraint](../keywords/new-constraint--csharp-reference-.md) Qualifies a type parameter.  
  
-   [sizeof](../keywords/sizeof--csharp-reference-.md) Obtains the size of a type.  
  
-   [typeof](../keywords/typeof--csharp-reference-.md) Obtains the **System.Type** object for a type.  
  
-   [true](../keywords/true--csharp-reference-.md)  
  
    -   [true Operator](../keywords/true-operator--csharp-reference-.md) Returns the boolean value true to indicate true and returns false otherwise.  
  
    -   [true Literal](../keywords/true-literal--csharp-reference-.md) Represents the boolean value true.  
  
-   [false](../keywords/false--csharp-reference-.md)  
  
    -   [false Operator](../keywords/false-operator--csharp-reference-.md) Returns the Boolean value true to indicate false and returns false otherwise.  
  
    -   [false Literal](../keywords/false-literal--csharp-reference-.md) Represents the boolean value false.  
  
-   [stackalloc](../keywords/stackalloc--csharp-reference-.md) Allocates a block of memory on the stack.  
  
 The following keywords, which can be used as operators and as statements, are covered in the [Statements](../keywords/statement-keywords--csharp-reference-.md) section:  
  
-   [checked](../keywords/checked--csharp-reference-.md) Specifies checked context.  
  
-   [unchecked](../keywords/unchecked--csharp-reference-.md) Specifies unchecked context.
