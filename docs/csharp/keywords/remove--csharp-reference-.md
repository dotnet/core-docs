# remove (C# Reference)
The `remove` contextual keyword is used to define a custom event accessor that is invoked when client code unsubscribes from your [event](../keywords/event--csharp-reference-.md). If you supply a custom `remove` accessor, you must also supply an [add](../keywords/add--csharp-reference-.md) accessor.
