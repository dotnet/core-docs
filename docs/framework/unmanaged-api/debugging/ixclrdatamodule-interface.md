# IXCLRDataModule Interface
Provide methods for querying information about a loaded module.

## Methods
|Method                    |Description                               |
|--------------------------|------------------------------------------|
|GetMethodDefinitionByToken|Get a method definition by metadata token.|
|GetVersionId              |Get the module's version ID.              |
