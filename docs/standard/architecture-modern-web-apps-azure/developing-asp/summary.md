# Summary

ASP&period;NET Core is a cross-platform, open-source framework for building modern cloud-optimized web applications. ASP&period;NET Core apps are lightweight and modular, with built-in support for dependency injection, enabling in greater testability and maintainability. Combined with MVC, which supports building modern web APIs in addition to view-based apps, ASP&period;NET Core is a powerful framework with which to build enterprise web applications.
